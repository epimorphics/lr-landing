Open Data

Land Registry publishes the following public datasets on GOV.UK as part of our commitment to the Governmentâ€™s priorities of economic growth and data transparency:

  • House Price Index background tables updated monthly, data available from January 1995.
  • Price paid data updated monthly, data available from 1995.
  • Transaction data updated monthly, data available from December 2011.

These datasets are provided in comma-separated value (csv) and linked data formats, with Price Paid Data also available as a text file. These datasets can be downloaded in full by accessing our Public Data.

Bespoke reports

Price Paid Data and House Price Index customers who do not require the full dataset can run bespoke reports with the following tools:

  • Search the house price index
  • Build a price paid data report
  • Build a standard price paid data report

Search the House Price Index

Use Search the House Price Index to find:

  • average price
  • sales volumes
  • property type
  • for a specified location
  • over a defined period of time

Our House Price Index compares average house price today to what it was in January 1995, with the index set then at 100. It includes figures at national, regional, county and Local Authority level. For more information read About the House Price Index.

Price Paid Data Report Builder

Use Price Paid Data Reports to build bespoke reports using our detailed Price Paid Data. Reports can be based on location, tenure, price paid or property type over a defined period of time.

Our Price Paid Data tracks the residential property sales in England and Wales that are lodged with us for registration. The dataset is a reliable source of house price information and consists of more than 24 million definitive records dating back to January 1995. For more information on this dataset and what it does and doesn't include, read about the Price Paid Data.

We strive to ensure that our public data is as accurate as possible but cannot guarantee that it is free from errors or fit for your purpose or use. Reports are based on data collected at the time a property transaction is registered with us and will not necessarily be up to date with the most recent information. Read "how accurate is the data?" for more information.

Further information

  • YouTube video tutorial on using the report builder.

Standard Report Builder

The Standard Report Builder is provided using our Price Paid Data. Use this tool to build bespoke reports that are aggregated price data. This service allows you to configure a report for various geographical areas in England and Wales, from the country level down to postcode sectors over a defined period of time.

Linked Data

We also publish this data in linked data format, aiming for five-star rating

If you would like to try out your own SPARQL queries against our linked data publications, use this SPARQL form. We have included some example queries with the form, and you can generate the output as a table, in plain text, as a JSON file or in XML.

If you want to see the SPARQL query that is used to generate the reports in the HPI and PPD reporting tools described above, you can do this by selecting the 'view SPARQL query' actions from the 'download data' view in each search tool. You can even amend and re-run the SPARQL query as required.
